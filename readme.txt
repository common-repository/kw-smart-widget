Kapitalwise Smart Widget Wordpress Plugin

== Description ==
This is a WordPress plugin for adding a smart widget of Kapitalwise on any website

== Installation ==

This plugin requires a Kapitalwise account, please create an account at https://kapitalwise.com

Please read the terms and conditions at https://www.kapitalwise.com/TermsAndConditions.html

Steps to Install Smart Widget Plugin

1. Download the Plugin as a Zip-file
2. Go in your WordPress Dashboard to ‘Plugins’ -> ‘Install’
3. Click on Upload
4. Click on ‘Choose file’ and choose the downloaded Zip-file
5. Click on ‘Install’
6. Activate the plugin through the ‘Plugins’ menu in WordPress
7. Now you can go to the Settings-Page on ‘Settings’ -> ‘Kapitalwise Smart Widget’
8. Enter the API key from the Kapitalwise Smart Widget Campaign page
9. Add the shortcode [kwsw_smart_widget] on any webpage to add the Smart Widget
10. Click on the Preview button to see the Kapitalwise Smart Widget on the webpage
